OWS Context 1.0 Conformance Test Suite

This executable test suite (ETS) verifies the conformance of context documents with respect to the OGC OWS Context Conceptual Model specification, version 1.0 (OGC 12-080r2). A context document specifies a fully configured service set which can be exchanged among clients supporting the standard; it was created to allow a set of configured information resources to be passed between applications primarily as a collection of services.
